### Use cases

Search by track name <br />
Filter by liked tracks <br />
Like and unlike tracks <br />
Play tracks <br />
Upload tracks <br />
Remove your uploaded tracks <br />

User authentication, login, signup, logout <br />

### Technologies

React <br />
React Context <br />
AWS Amplify <br />
React Material UI <br />

Download and run commands <br />

 2 - npm start
